    集合¶

    a = {
    'x' : 1,
    'y' : 2,
    'z' : 3
}

b = {
    'w' : 10,
    'x' : 11,
    'y' : 2
}

    为了寻找两个字典的相同点，可以简单的在两字典的 keys() 或者 items() 方法返回结果上执行集合操作。比如：

    # Find keys in common
a.keys() & b.keys() # { 'x', 'y' }
# Find keys in a that are not in b
a.keys() - b.keys() # { 'z' }
# Find (key,value) pairs in common
a.items() & b.items() # { ('y', 2) }

    从一个集合中获得最大或者最小的 N 个元素列表

    import heapq
nums = [1, 8, 2, 23, 7, -4, 18, 23, 42, 37, 2]
print(heapq.nlargest(3, nums)) # Prints [42, 37, 23]
print(heapq.nsmallest(3, nums)) # Prints [-4, 1, 2]

    队列¶

    from collections import deque
d=deque(maxlen=5)#设置长度
>>> q = deque() #不设置长度
>>> q.append(1)
>>> q.append(2)
>>> q.append(3)
>>> q
deque([1, 2, 3])
>>> q.appendleft(4)
>>> q
deque([4, 1, 2, 3])
>>> q.pop()
3
>>> q
deque([4, 1, 2])
>>> q.popleft()
4

    在队列两端插入或删除元素时间复杂度都是 O(1) ，区别于列表，在列表的开头插入或删除元素的时间复杂度为 O(N) 。

    装饰器¶

    在装饰器中使用functool.wraps

    import functools

def uppercase(func):
    @functools.wraps(func)
    def wrapper():
        return func().upper()
    return wrapper

    @uppercase
def greet():
    '''return a friendly greeting'''
    return 'hello'

    关于import _init_.py 模块(module) 包(package)¶

    目录结构

    └─demo
        a.py
        b.py
        __init__.py

    因为有 _init_.py, 所以demo是个package

    • 无论是 import demo 或者 from demo import xxx, init.py文件中的语句都会执行
    • 可以被导入的文件叫module, 可以被导入的目录叫做包package

    class WhoSay:
    def __init__(self,who):
        self.who =who

    def say(self):
        self.who.say()
